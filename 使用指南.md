# glownight blog 项目使用指南

## 📋 项目概览

**glownight blog** 是一个基于 Astro 框架构建的个人博客网站，使用 astro-pure 主题。

- **项目地址**：https://glownight
- **技术栈**：Astro 5.x + TypeScript + UnoCSS + MDX
- **许可证**：博客内容采用 CC BY-NC-SA 4.0 协议

---

## 🚀 快速开始

### 环境要求

- **Node.js**：>= 18.20.8（推荐 20.x 或 22.x LTS 版本）
- **包管理器**：npm / bun / pnpm

### 安装步骤

#### 1. 安装依赖

```bash
npm install
```

#### 2. 启动开发服务器

```bash
npm run dev
```

启动成功后，访问 `http://localhost:4321/` 查看网站。

#### 3. 构建生产版本

```bash
npm run build
```

## ✍️ 如何写博客

### 方法一：使用命令行工具（推荐）

项目提供了便捷的命令行工具来创建新文章：

```bash
# 创建普通文章
npm run new "文章标题"

# 创建中文文章
npm run new -l zh "文章标题"

# 创建草稿文章（不会发布）
npm run new -d "文章标题"

# 使用 MDX 格式（支持 React 组件）
npm run new -m "文章标题"

# 在文件夹中创建文章
npm run new -f "文章标题"
```

**完整选项说明**：
- `-l, --lang <en|zh>` - 设置语言
- `-d, --draft` - 创建为草稿
- `-m, --mdx` - 使用 MDX 格式
- `-f, --folder` - 在文件夹中创建文章
- `-h, --help` - 显示帮助信息

### 方法二：手动创建文章

在 `src/content/blog/` 目录下创建 `.md` 或 `.mdx` 文件。

## 📝 文章格式

### Frontmatter（前置元数据）

```markdown
---
title: 文章标题
description: '文章描述，用于 SEO 和列表展示'
publishDate: 2024-01-27 10:00:00
updatedDate: 2024-01-28 15:00:00  # 可选
language: '中文'  # 可选
tags:
  - 标签1
  - 标签2
  - 标签3
draft: false  # 可选，默认为 false
comment: true  # 可选，是否启用评论，默认为 true
heroImage:  # 可选，封面图
  src: '/images/cover.jpg'
  alt: '封面图描述'
---

# 正文内容

这里是文章的正文，使用 Markdown 语法编写。
```

### 字段说明

**必填字段**：
- `title` - 文章标题（最多 60 字符）
- `description` - 文章描述（最多 160 字符）
- `publishDate` - 发布日期（格式：YYYY-MM-DD HH:MM:SS）

**可选字段**：
- `updatedDate` - 更新日期
- `language` - 语言（如 "中文"、"English"）
- `tags` - 标签数组（自动去重并转小写）
- `draft` - 是否为草稿（true 不会发布）
- `comment` - 是否启用评论（默认 true）
- `heroImage` - 封面图配置

## 💡 写作技巧

### 1. 数学公式

使用 LaTeX 语法：

```markdown
行内公式：$E = mc^2$

块级公式：
$$\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}$$
```

### 2. 代码高亮

支持多种语言：

```markdown
\```python
def hello():
    print("Hello, World!")
\```
```

### 3. 图片缩放

添加 `zoomable` 类启用缩放效果：

```markdown
![图片描述](/path/to/image.jpg){.zoomable}
```

### 4. 代码块功能

- `diff` 标记增删行
- `highlight` 高亮特定行
- 自动添加复制按钮
- 显示语言标签

### 5. MDX 高级用法

如果使用 `.mdx` 格式，可以嵌入 React 组件：

```markdown
---
title: MDX 示例
---

import { Button } from '../components/Button.astro'

# 使用组件

<Button>点击我</Button>
```

## 📤 发布文章

1. 写完文章后，确保 `draft: false`（或删除 draft 字段）
2. 运行 `npm run dev` 预览效果
3. 确认无误后提交到 Git 仓库
4. 运行 `npm run build` 构建生产版本
5. 部署到服务器（如 Vercel）

## ⚠️ 常见问题

### 问题 1：Node.js 版本不兼容

**错误信息**：
```
Node.js v18.20.5 is not supported by Astro!
Please upgrade Node.js to a supported version: ">=18.20.8"
```

**解决方案**：

#### 方法一：升级 Node.js（推荐）

1. 访问 [Node.js 官网](https://nodejs.org/) 下载最新 LTS 版本
2. 安装完成后重启终端
3. 重新安装依赖：`npm install`

#### 方法二：使用 nvm（Node Version Manager）

```bash
# 安装 Node.js 20.x
nvm install 20
nvm use 20

# 重新安装依赖
npm install
npm run dev
```

### 问题 2：依赖安装失败

**解决方案**：

```bash
# 清理缓存和依赖
rm -rf node_modules package-lock.json

# 重新安装
npm install
```

### 问题 3：端口被占用

**解决方案**：

修改 `astro.config.ts` 中的端口配置，或在命令中指定端口：

```bash
npx astro dev --port 3000
```
